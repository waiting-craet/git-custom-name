### <font style="color:rgb(0, 0, 0);">项目开发指令：个人博客系统前端实现</font>
#### <font style="color:rgb(0, 0, 0);">一、项目背景与目标</font>
<font style="color:rgb(0, 0, 0);">开发一套基于 HTML+CSS+JavaScript 的个人博客系统前端界面，需配合后端 Flask 框架及 SQLite 数据库，实现完整的博客浏览、发布、用户交互功能，同时满足操作反馈、表单验证等技术要求，最终呈现简洁易用、交互友好的博客体验。</font>

#### <font style="color:rgb(0, 0, 0);">二、技术栈与开发规范</font>
1. **<font style="color:rgb(0, 0, 0) !important;">核心技术</font>**<font style="color:rgb(0, 0, 0);">：严格使用 HTML5 + CSS3 + 原生 JavaScript 实现（不依赖第三方前端框架）；</font>
2. **<font style="color:rgb(0, 0, 0) !important;">后端配合</font>**<font style="color:rgb(0, 0, 0);">：需适配 Flask 后端框架的模板渲染逻辑（如支持 Flask 模板变量、表单提交 action 指向 Flask 路由）；</font>
3. **<font style="color:rgb(0, 0, 0) !important;">数据库交互</font>**<font style="color:rgb(0, 0, 0);">：前端无需直接操作 SQLite，仅需通过表单提交、API 请求（若有）与后端交互数据；</font>
4. **<font style="color:rgb(0, 0, 0) !important;">开发工具</font>**<font style="color:rgb(0, 0, 0);">：基于你的代码生成能力，输出可直接运行的静态文件及关联逻辑。</font>

#### <font style="color:rgb(0, 0, 0);">三、前端页面与功能清单</font>
<font style="color:rgb(0, 0, 0);">需实现以下核心页面，要求页面风格统一（建议采用简洁现代风，配色以浅色为主，突出文字内容），包含完整导航与页脚：</font>

1. **<font style="color:rgb(0, 0, 0) !important;">首页（index.html）</font>**
    - <font style="color:rgb(0, 0, 0);">顶部导航栏：包含博客名称、首页、分类、关于我、登录 / 注册入口（登录后显示用户名 / 退出按钮）；</font>
    - <font style="color:rgb(0, 0, 0);">主体内容：博客文章列表（每篇含标题、摘要、发布时间、作者、阅读量、分类标签），支持分页；</font>
    - <font style="color:rgb(0, 0, 0);">侧边栏：热门文章、分类列表、标签云；</font>
    - <font style="color:rgb(0, 0, 0);">页脚：版权信息、联系方式。</font>
2. **<font style="color:rgb(0, 0, 0) !important;">文章详情页（post.html）</font>**
    - <font style="color:rgb(0, 0, 0);">文章主体：标题、完整内容（支持基础排版，如段落、换行）、发布时间、作者、分类标签；</font>
    - <font style="color:rgb(0, 0, 0);">交互功能：评论区（显示已有评论、评论表单）、点赞按钮；</font>
    - <font style="color:rgb(0, 0, 0);">导航：返回首页、上一篇 / 下一篇文章链接。</font>
3. **<font style="color:rgb(0, 0, 0) !important;">发布 / 编辑文章页（edit_post.html）</font>**
    - <font style="color:rgb(0, 0, 0);">表单包含：标题输入框（必填）、内容文本域（必填，支持简单换行）、分类下拉选择（必填）、标签输入框（可选，多标签用逗号分隔）；</font>
    - <font style="color:rgb(0, 0, 0);">提交按钮（发布 / 更新）、取消按钮（返回文章列表）。</font>
4. **<font style="color:rgb(0, 0, 0) !important;">登录页（login.html）</font>**
    - <font style="color:rgb(0, 0, 0);">表单包含：用户名 / 邮箱输入框（必填）、密码输入框（必填，带密码可见 / 隐藏切换）；</font>
    - <font style="color:rgb(0, 0, 0);">提交按钮、注册跳转链接。</font>
5. **<font style="color:rgb(0, 0, 0) !important;">注册页（register.html）</font>**
    - <font style="color:rgb(0, 0, 0);">表单包含：用户名（必填，验证唯一性提示）、邮箱（必填，验证格式）、密码（必填，长度≥6 位）、确认密码（必填，与密码一致）；</font>
    - <font style="color:rgb(0, 0, 0);">提交按钮、登录跳转链接。</font>
6. **<font style="color:rgb(0, 0, 0) !important;">个人中心页（profile.html）</font>**
    - <font style="color:rgb(0, 0, 0);">显示用户信息：用户名、注册时间；</font>
    - <font style="color:rgb(0, 0, 0);">我的文章列表：仅显示当前用户发布的文章，带编辑 / 删除按钮。</font>

#### <font style="color:rgb(0, 0, 0);">四、核心技术要求实现细节</font>
1. **<font style="color:rgb(0, 0, 0) !important;">Flash 消息反馈功能</font>**
    - <font style="color:rgb(0, 0, 0);">需求：显示后端返回的操作结果（如 “登录成功”“发布文章失败：标题不能为空”“评论提交成功”）；</font>
    - <font style="color:rgb(0, 0, 0);">实现：</font>
        * <font style="color:rgb(0, 0, 0);">在页面顶部 / 导航下方预留消息展示区域，支持动态插入；</font>
        * <font style="color:rgb(0, 0, 0);">区分消息类型：成功（绿色背景 + 对勾图标）、错误（红色背景 + 叉号图标）、警告（黄色背景 + 感叹号）；</font>
        * <font style="color:rgb(0, 0, 0);">交互：消息显示后 3 秒自动消失，或支持手动点击关闭；</font>
        * <font style="color:rgb(0, 0, 0);">适配 Flask：通过获取 Flask 模板变量（如</font>`<font style="color:rgb(0, 0, 0);">{{ get_flashed_messages() }}</font>`<font style="color:rgb(0, 0, 0);">）渲染消息。</font>
2. **<font style="color:rgb(0, 0, 0) !important;">表单数据验证与错误处理</font>**
    - <font style="color:rgb(0, 0, 0);">实时前端验证（提交前）：</font>
        * <font style="color:rgb(0, 0, 0);">非空验证：所有 “必填” 字段为空时，输入框边框标红，下方显示 “此字段不能为空” 提示；</font>
        * <font style="color:rgb(0, 0, 0);">格式验证：邮箱需匹配</font>`<font style="color:rgb(0, 0, 0);">xxx@xxx.xxx</font>`<font style="color:rgb(0, 0, 0);">格式，不匹配时提示 “请输入正确邮箱格式”；</font>
        * <font style="color:rgb(0, 0, 0);">一致性验证：注册页 “确认密码” 与 “密码” 不一致时，提示 “两次密码输入不一致”；</font>
        * <font style="color:rgb(0, 0, 0);">长度验证：密码长度＜6 位时，提示 “密码长度至少 6 位”。</font>
    - <font style="color:rgb(0, 0, 0);">后端返回错误处理：</font>
        * <font style="color:rgb(0, 0, 0);">表单提交后，若后端返回错误（如用户名已存在），前端需在对应字段下方显示后端返回的错误信息；</font>
        * <font style="color:rgb(0, 0, 0);">保留用户已输入的正确内容，避免重新填写。</font>
3. **<font style="color:rgb(0, 0, 0) !important;">基础交互体验优化</font>**
    - <font style="color:rgb(0, 0, 0);">导航栏滚动时固定在顶部；</font>
    - <font style="color:rgb(0, 0, 0);">按钮、链接 hover 状态有样式变化（如颜色加深、微小缩放）；</font>
    - <font style="color:rgb(0, 0, 0);">表单提交时添加 “加载中” 状态（禁用按钮 + 显示加载图标），防止重复提交；</font>
    - <font style="color:rgb(0, 0, 0);">评论提交后无需刷新页面，动态添加到评论列表。</font>
